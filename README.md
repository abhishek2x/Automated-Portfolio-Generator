## Getting Started

- Fork and Clone the repo <br/>
  `git clone https://github.com/<your-username>/VITB-folder.git`
- Change the directory<br/>
  `cd VITB-folder`
- Install node dependencies<br/>
  `npm i` or `npm install` or `yarn`
- Start the Server<br/>
  `npm start` or `yarn start`
